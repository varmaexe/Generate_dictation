# How to Run — Dictation Generator

Hey Shravya! 👋 Just two steps and you're all set forever.

---

## Step 1 — Install the required library (one time only)

Press **Command (⌘) + Space**, type **Terminal**, and press Enter.

Paste this and press Enter:

```
pip3 install openpyxl
```

Wait for it to finish. You'll see the `$` prompt again when it's done. Close Terminal.

---

## Step 2 — Run the tool

Put both `generate_dictation.py` and `run.command` in the same folder (e.g. your Desktop).

Double-click **run.command** — that's it! 🎉

A terminal window will open, ask you a couple of simple questions, and when it's done your Excel file will open automatically.

> **First time only:** Mac may ask *"Are you sure you want to open this?"* — click **Open**.

---

## Every time after that

Just double-click **run.command**. No typing needed.
